				
1. Write a function to compute divide by zero and use try/except to catch the exceptions.
Ans:
def divide(a,b):
    try:
        result=a//b
        print("Your answer is:",result)
    except ZeroDivisionError:
        print("You are dividing by zero")
divide(3,2)
divide(6,0)

3. Write python program to check  that given email address is valid or not?
Ans:
import re
regex='^[a-z0-9]+[\._]?[a-z0-9]+[@]\w+[.]\w+$'
def isvalid(email):
    if(re.search(regex,email)):  
        print("Valid Email")  
    else:  
        print("Invalid Email")  
isvalid("bpriyanka513@gmail.com")

6. Write a python program to generate Fibonacci Numbers upto 100 using generator
Ans:
def fib(mymax):
    n1,n2=1,2
    while True:
        n3=n1+n2
        if n3<mymax:
            yield n3
            n1=n2
            n2=n3
        else:
            break
generator=fib(101)
print(next(generator))
